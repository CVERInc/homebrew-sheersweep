# homebrew-sheersweep

Homebrew tap for [sheersweep](https://github.com/CVERInc/sheersweep) — the Mac cleaner you can read.

```bash
brew install CVERInc/sheersweep/sheersweep
```

Then:

```bash
sheersweep --dry-run   # preview, deletes nothing
sheersweep             # real run (prompts once for sudo)
```
